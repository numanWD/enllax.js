Enllax.js

Enllax.js is an ultra-lightweight parallax Library in VanillaJS, with foreground and background images on Scroll.

View on GitHub Download .zip Download .tar.gz

Background Parallax

Default behaviour.

Code:

enllax('#js-content-1');

Background and Foreground Parallax

We have a background parallax with a ratio .6 of the normal scroll.

If you scroll 100px the image will scroll 60px.

We have a foreground horizontal parallax

Code

    //Background
    enllax('.content--2', {
        ratio: .6
    });

    //Foreground
    enllax('.foreground--1', {
        ratio: -.8,
        type: 'foreground',
        direction: 'horizontal'
    });

Background Static

ratio: 1

Code:

    enllax('#js-content-2', {
        ratio: 1
    });

Multi Parallax - Rocket

In this example we have implemented one background and 3 foreground parallax.

Code:

    enllax('#js-rocket-1', {
            ratio: 0.1
    });
    enllax('#js-rocket-fg-1', {
        ratio: 0.1,
        type: 'foreground'
    });
    enllax('#js-rocket-fg-2', {
        ratio: 0.15,
        type: 'foreground'
    });
    enllax('#js-rocket-fg-3', {
        ratio: 0.6,
        type: 'foreground'
    });

How to use it:

Enllax.js doesn't have any dependency.
To make it work just

Load the Enllax.js library in the footer


    <script src="enllax.js""></script">

After the library open a script tag and call the function


    <script src="enllax.js"">
        enllax('.selector');
        enllax('#selector');
    </script">

You can use any of the css selectors.

  • #{id} - Id
  • .{class} - Class
  • ...

As second argument of the function, you can pass an object with specific parameters, by default the values are:


    {
        ratio: 0.5,
        type: 'background', // foreground
        direction: 'vertical' // horizontal
    }
